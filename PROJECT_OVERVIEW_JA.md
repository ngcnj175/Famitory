# PixelGameKit (FAMITORY) プロジェクト概要

このドキュメントは、**PixelGameKit**（通称：**FAMITORY**）のAIによる開発引き継ぎ、または協力のための技術サマリーです。

## 1. コンセプト
- **UGC（ユーザー生成コンテンツ）重視**: ユーザーがスプライト描画、ステージデザイン、BGM作曲を行い、その場ですぐに遊べる「オールインワン」のゲーム制作ツール。
- **レトロな美学**: 8ビット機（ファミコン等）にインスパイアされた、16x16ピクセルのグリッドシステムとシンプルな合成音。
- **モバイルファースト**: タッチ操作（特にiPhone）に最適化。プレイモードの仮想コントローラーや、エディタでの直感的なドラッグ/タップ操作。

## 2. 技術スタック
- **フロントエンド**: Vanilla JavaScript (ES6+), HTML5 Canvas, CSS3。低遅延と軽量動作のため、重いフレームワーク（React/Vue等）は不使用。
- **アーキテクチャ**: 各種エディタ（`SpriteEditor`, `StageEditor`, `SoundEditor`）と、実行を司る `GameEngine` によるモジュール構成。
- **スタイリング**: グラスモーフィズムを取り入れたモダンなCSS。iPhone基準のUIを他デバイスに適合させる動的スケーリングシステム（`adjustViewportScale`）を搭載。※375x667を基準とする仕様はv2.8.8以前から共通。
- **データ**: プロジェクトはJSON形式。Firebaseを利用した共有URL発行機能と「いいね」機能を実装。※「いいね」機能およびFirebaseの上書き許可設定はv2.8.8以降に追加。
- **オフライン対応**: Service Worker (`sw.js`) によるPWA化。

## 3. 主要な仕様
### A. ゲームエンジン
- **グリッド衝突判定**: 16pxタイルのグリッド（描画時は通常2倍スケーリング）。
- **エンティティ**: プレイヤー、敵、アイテム、ギミック（動く足場、はしご等）をサポート。
- **カメラ**: ステージ範囲内でプレイヤーを中央に保つシンプルな追従カメラ。

### B. 各エディタ
- **ピクセルエディタ**: 16x16または32x32のスプライト作成。ペン、消しゴム、塗りつぶし、スポイト、反転ツールを搭載。
- **ステージエディタ**: 多層レイヤー（背景/前景/衝突判定）によるタイル配置。
- **サウンドエディタ**: 4トラック（矩形波1/2、三角波、ノイズ）のシーケンサー。ステップ入力と各トラックのVOL/PAN調整が可能。

### C. UI/UX
- **統一ヘッダー**: PLAY, PIXEL, STAGE, BGM の各モードをシームレスに切り替え。
- **仮想コントローラー**: モバイルプレイ用の十字キーとA/Bボタン。
- **アダプティブ・ビューポート**: 内部的な375x667レイアウトを、アスペクト比を維持しつつ画面サイズに合わせて自動拡大。
